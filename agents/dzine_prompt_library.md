## Usage Notes

- **Thumbnail prompts** (T01-T03): Use Txt2Img mode, 16:9 ratio, 2K quality
- **Ray scene prompts** (R01-R07): Use Consistent Character mode ONLY, select Ray, canvas (16:9) ratio
- **Product prompts** (P01-P03, L01-L06, D01-D03, M01-M03): Use Txt2Img with reference image when available
- **Background prompts** (B01-B03): Use Txt2Img, no reference, 16:9 ratio
- All prompts are under 1800 chars to fit Txt2Img/CC limits
- Replace `{product_name}` at runtime with actual product name from products.json
