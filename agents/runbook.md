# Runbook (1 long video/day)

Daily flow:

1. Curation: pick one high-opportunity topic ($100+), fill curation template.
2. Script: generate one long script (8-12 min), review for accuracy.
3. Production: generate avatar/voice assets and edit one long video.
4. Checklist: confirm disclosures and policy compliance.
5. Upload: publish with affiliate disclosures.

Time budget:

- Curation: 20 min
- Script: 20 min
- Production: 45-60 min
- Review + Upload: 15 min

File naming:

- YYYY-MM-DD_topic_long_v1.mp4

Storage:

- Keep raw scripts and final video URLs in a spreadsheet.
