# Affiliate Compliance

Disclosure must be visible in video description and generated metadata.

## Link Rules

- Allow Amazon direct links and SiteStripe short links.
- Reject non-transparent re-shorteners that hide destination.
- Ensure tracking tag appears exactly once when constructing links.

## Publication Rules

- Include disclosure text in description and pinned-comment draft.
- Keep claim language non-deceptive and non-absolute.
