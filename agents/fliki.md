# Agent: FlikiAI (Avatar + Shorts)

Goal: Produce a vertical 9:16 Shorts video with AI avatar using FlikiAI.

Inputs:

- Final script (from Script Agent)
- Avatar selection
- Music/SFX selection (royalty-safe)

Steps:

1. Open FlikiAI and create a new video.
2. Set format to 9:16 (Shorts).
3. Choose an AI avatar (consistent across videos).
4. Paste SCRIPT into the editor.
5. Add ON-SCREEN TEXT lines as captions or title cards.
6. Add background music (low volume).
7. Add subtle SFX if needed (whoosh/click).
8. Add disclosure line in a small lower-third or end card.
9. Export MP4.

Export settings:

- 1080x1920
- 30 fps
- H.264

Output checklist:

- 20 seconds or less
- Clear audio
- Text readable on mobile
