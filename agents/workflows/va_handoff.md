# SOP: VA Handoff (Editing + Upload)

Goal: Allow a VA to assemble and upload videos consistently.

Inputs:

1. Final script
2. Voiceover segments
3. Visual assets
4. Edit guide
5. Description template with disclosures

Steps:

1. Import voiceover and assets into the editing tool.
2. Assemble scenes in the exact order of the edit guide.
3. Add captions and ensure readability.
4. Add background music at low volume.
5. Export at highest quality.
6. Upload with title, description, and disclosures.
7. Notify Lead with the final URL.
