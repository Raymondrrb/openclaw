# Visual Guidelines

1. Use a consistent avatar and style.
2. Prefer licensed stock or AI visuals you generated.
3. Match each scene to a specific claim in the script.
4. Avoid copyrighted logos unless allowed.
5. Keep on-screen text short and high contrast.
6. Track each asset in `asset_manifest.md` with source URL + usage note.
7. Use only assets marked `APPROVED` for final export.
