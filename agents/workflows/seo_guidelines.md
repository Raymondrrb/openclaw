# SEO Guidelines (Rayviews)

## Goal

Maximize qualified clicks (CTR) and retention for a Top 5 buying-intent audience.

## Workflow (mandatory)

1. Build keyword set:

- Primary: category + year intent (`best <category> 2026`, `top 5 <category>`).
- Secondary: use-case + price intent (`for <use-case>`, `under $X`, `vs`).
- Validation sources: Google Trends, YouTube search autosuggest, top competing titles.

2. Score title candidates:

- Keep 3 options before final lock.
- Prefer titles with clear buyer intent + concrete object (`Top 5 Portable Monitors 2026`).
- Keep the primary keyword in the first 60 characters.
- Avoid generic clickbait (`you won't believe`, `insane`, `shocking`).

3. Description structure:

- First 2 lines: primary keyword + value promise.
- Then product list with affiliate links.
- Then disclosures:
  - affiliate disclosure
  - "prices may change, check link for current price"
  - "at time of recording" for dynamic claims

4. Chapters:

- 8–12 chapters max, aligned with the actual timeline.
- Include product name + intent term, not filler labels.
- Example: `03:42 #3 Best for Battery Life`.

5. Tags and hashtags:

- 8–12 tags only, tightly relevant.
- Use 2–3 hashtags max in description footer.

## Hard Rules

- No misleading claims.
- No unverifiable superlatives ("best ever", "perfect", "#1 guaranteed").
- If volume signal is weak, choose specificity over broad keyword stuffing.

## Output checklist

- 3 title options + 1 chosen final title.
- Description ready-to-paste with links/disclosures.
- Chapters with timestamps.
- Tag list + hashtags.
