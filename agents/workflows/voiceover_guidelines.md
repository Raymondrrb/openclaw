# Voiceover Guidelines

1. Mandatory voice: `Thomas Louis` in ElevenLabs.
2. Generate in segments of 60-90 seconds.
3. Keep one consistent model + settings preset across all episodes.
4. Keep pacing conversational but efficient.
5. Avoid hype or unverifiable claims.
6. Target script size for 8-12 min: ~1,300-1,750 words (about 8k-11k chars).
7. Save each segment with numbered filenames:
   - `vo_01_hook.wav`
   - `vo_02_criteria.wav`
   - `vo_03_rank5.wav`
   - `...`
